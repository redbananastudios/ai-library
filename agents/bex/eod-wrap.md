# EOD Wrap Template

**Posted by Bex to `#bex` at 17:00 weekdays.**

## Format

```
**Shipped today**
• [brand]: [outcome] — [link if applicable]
• ...

**In flight for tomorrow**
• [brand]: [task] — [owner: me or Viktor] — [due]
• ...

**New blockers** ({count})
• [owner] on [task] — blocker: [reason]
• ...

**Spend/commitments made today** (if any)
• [item] — £[amount] — [approved by Peter at link]

**Tomorrow's headline**
[One sentence — what matters most about tomorrow's plan]
```

## Rules

- **Skip empty sections**
- **"Shipped today"** = real outcomes. Posted, published, delivered, merged. Not "drafted" or "started"
- **"In flight"** = anything that will continue or land tomorrow. Max 3-5 items total
- **"New blockers"** only — don't re-list blockers carried over from this morning's brief
- **"Spend/commitments"** — only list if there was approved spend today. Include amount and approval link
- **"Tomorrow's headline"** — one sentence. Sets the frame for tomorrow's morning brief

## Reading order Bex follows to generate this

1. ClickUp: tasks closed today, filter for shipped
2. ClickUp: tasks open with due-date tomorrow or in-progress
3. ClickUp: tasks moved to Blocked today
4. `#approvals`: approved threads from today
5. Compose the wrap

## Tone reminder

- No "Great day team!" — keep it factual
- No emojis (except in approval reactions)
- If today was bad, say so: *"Marley ads underperformed (CTR 0.4% vs 1.2% target)"*
- If today was great, say so plainly: *"Willow Amazon listing live, 3 orders in first 4 hours"*
