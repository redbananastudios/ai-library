# Changelog - Firecrawl Web Scraper

## 0.1.0 (2026-04-09)

- Initial creation
- Single-page scrape to markdown/HTML
- Full-site crawl with configurable limits
- Batch scrape across multiple URLs
- AI-powered structured data extraction with JSON schemas
- URL mapping (discover all URLs on a site)
- Web search with scrape integration
- Support for Zod schema validation (JS) and JSON Schema
- Environment variable API key configuration
