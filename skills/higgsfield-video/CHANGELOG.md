# Changelog - higgsfield-video

## [1.0.0] - 2026-04-13

### Added
- Initial skill creation with comprehensive Higgsfield.ai video generation support
- 9 video models: Cinema Studio, Kling 3.0, Kling 2.5 Turbo, Seedance 2.0, Veo 3.1 (UGC Builder), WAN 2.5, Sora 2, Minimax Hailuo 02, UGC Factory
- 4 production workflows: Cinema Studio, Standard Video, UGC Builder, Multi-Scene Storyboard
- Scene definition template with camera body/lens selection, camera movements (70+ presets), lighting, duration, aspect ratio
- Browser automation workflow using browser-use CLI or Playwright MCP
- Scene-by-scene generation with ffmpeg stitching support
- Integration with nano-banana-realism-engine for cinematic prompt engineering
- 6 reference documents: video models, camera controls, Cinema Studio guide, UGC guide, scene template, stitching guide
- Model auto-selection guide for UGC, social, cinematic, testimonial workflows
- Output directory: `./higgsfield-videos/`
